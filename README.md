# For You, Anju 🌸

A personal love letter website — a little reminder of how loved you are.

## 🌷 Features

- Full-screen scroll-snapping photo sections
- Frosted glass text cards with fade-in animations
- Floating hearts, sparkles, and petals
- Custom heart cursor on desktop
- Mobile-responsive design
- Love button popup

## 📁 Structure

```
ANU-PER/
├── index.html        ← Everything inline (HTML + CSS + JS)
├── images/
│   ├── photo1.jpg    ← Close-up eyes
│   ├── photo2.jpg    ← Couple selfie
│   ├── photo3.jpg    ← Instagram filter
│   ├── photo4.jpg    ← Baby photo
│   └── photo5.jpg    ← B&W studio
└── README.md
```

## 📸 Adding More Photos

1. Add photos to the `images/` folder (e.g., `photo6.jpg`, `photo7.jpg`, etc.)
2. In `index.html`, copy any `<section>` block and update the background image URL and quote text.

## 🚀 Deploy on GitHub Pages

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to **Deploy from a branch → main → / (root)**
4. Click Save — your site will be live in ~1 minute

## 🚀 Deploy on Render

1. Go to [render.com](https://render.com) → **New → Static Site**
2. Connect your GitHub repo
3. Set **Publish directory** to `/`
4. Click **Create Static Site**
